JavaScript a lot of work behind the seen So the browser doesn’t understand javascript directly.The browser only understands 0 and 1 language . Statements in Binar format, we can’t convert our whole JavaScript into bits easily thats why need javascript engine. Example V8 is javascript engine of chrome browser.When a JavaScript file is loaded into the browser, JavaScript Engine executes that file line by line from top to bottom.Then will parse the code line by line and convert that code into the machine code Binary.Now browser can understand this machine code and behave accordingly.

Single-threaded language is Javascript that means only execute one task at a time.Event loop has the ability to process multiple requests asynchronously. Event Loop work by the Event Queue and Call Stack.Event loop is to constantly monitor the call stack and the message queue.Event Loop follow this step .When the Call Stack is empty, all of the functions complete their execution and they pop off the Call Stack.Once the Call Stack is empty, it moves the first item from the Event Queue to the Call Stack.Sometime call stack and the message queue may be empty but the event loop never stops checking.That's all of about Event Loop.
